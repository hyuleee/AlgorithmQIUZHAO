## 数据结构<br>
  1.一维<br>
    a. 基础：array 数组；linked list 链表<br>
    b. 高级：stack 栈；queue 队列；deque 双端队列；set 集合；map 映射<br>
 <br>
  2. 二维<br>
    a. tree 树；graph 图<br>
    b. 二叉搜索树 （红黑树，AVL）；heap 堆；Trie 字典树； disjoint set 并查集<br>
  <br>
  3. 特殊<br>
    a. Bitwise 位运算; BloomFilter 布隆过滤器<br>
    b. LRU Cache 缓存<br>
  <br>
